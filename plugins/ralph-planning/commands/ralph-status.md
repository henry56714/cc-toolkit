# /ralph-status 命令

显示当前 Ralph-Planning 循环的详细状态。

## 用法

```
/ralph-status
```

## 显示信息

- 循环状态（运行中/已停止）
- 当前迭代次数
- 任务完成进度
- 当前正在执行的任务
- 错误统计
- 是否有人工介入请求
